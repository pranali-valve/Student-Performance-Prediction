# 📊 Student Performance Prediction System

## 📌 Project Overview

The Student Performance Prediction System is an end-to-end Machine Learning project that predicts student marks based on multiple factors such as:

- Study Hours
- Attendance Percentage
- Previous Marks

The project uses a Linear Regression Machine Learning model and provides real-time predictions through a Streamlit web application.
